# ecommerce-backend
e-commmerce backend using spring boot, influx db & grafana setup

## List of api 
#### Customer:
 - Get Api (email))
 - Delete Api 
 - Update Api(email))
 - Post Api 

#### Product:
- List all product,  
- Delete Api 
- Update Api
- Post Api

#### Order:
- Post Api (CustomerId, ProductId)
- Get Product by Customer
